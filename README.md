# Gestión de Flota: Ingeniería Inversa y Modelado de Datos

Este proyecto demuestra la reconstrucción de una arquitectura de base de datos relacional a partir de datos desestructurados en planillas de cálculo.

## tecnologías Utilizadas
* **SQL Server:** Diseño de esquema, normalización y creación de Vistas complejas.
* **R (Tidyverse):** Limpieza de datos (Data Wrangling), anonimización y análisis estadístico.
* **Git:** Control de versiones.

## Desafío Técnico
El objetivo principal fue aplicar **Ingeniería Inversa** a un set de datos de flota para:
1. Recrear la lógica de negocio original.
2. Limpiar inconsistencias de formato (comillas, saltos de línea).
3. Implementar un modelo de seguridad mediante **anonimización de datos** (Masking y Jittering).
